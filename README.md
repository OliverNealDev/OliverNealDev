## 🚀 Featured Projects

### [Cyber Station](https://oliverneal.dev)
A futuristic station management game where passengers, facilities, trains, and ratings feed into each other to create a lively simulation.

**Built with:** `Unity 3D` `AI Navigation` `Finite State Machines` `Object Pooling` `Management Systems`

---

### [Tile Turfer](https://oliverneal.dev)
A top-down shooter prototype where map coverage directly affects offensive power. The player paints the environment to upgrade fire rate and weapon behaviour dynamically.

**Built with:** `Unity 2D` `Tilemaps` `Coroutines` `AI Navigation` `Prototype Design`

---

### [Minimalists](https://embitgames.itch.io/minimalists)
A minimalist RTS prototype built around interconnected structures, AI behaviour, and node-based strategy.

**Built with:** `Unity 3D` `Finite State Machines` `AI Behaviour` `Level Design`

---

### [Unicellular](https://oliverneal.dev)
An idle entity simulation handling thousands of interacting units using FSMs and optimisation techniques.

**Built with:** `Unity 2D` `FSMs` `Entity Simulation` `Performance Optimisation`

A team project developed as part of Teesside University's Beta Arcade module. I worked as one of the gameplay programmers in a team of 10, implementing core gameplay systems, controllers, widgets, and interactables.
